## 変更内容
- 左上のシールドアイコンを廃止し、一般ダッシュボードおよび管理者検証パネルで共通の「波（Waves）のSVGアイコン」に変更しました。
- 一般ダッシュボードの左上にもアプリロゴ「kawa-watch」および役割バッジ「一般ダッシュボード」を追加し、管理者検証パネルとのデザイン一貫性を確保しました。
- アラート履歴画面（AlertHistory.jsx）の左上にも同様に共通の「波アイコン ＋ kawa-watch ＋ アラート履歴バッジ」を追加し、統一的なヘッダーレイアウトを適用しました。
- 一般ダッシュボードに表示されるテキストやボタン（`Alert History`、`Verification Mode`、`Admin Login`）を日本語（「アラート履歴」「検証モード」「管理者ログイン」）に統一しました。
- アラート履歴画面（AlertHistory.jsx）の表示（`Alert History`、`Back to Dashboard`、`No alerts found.`）および警告レベルバッジも、日本語（「アラート履歴」「ダッシュボードへ戻る」「現在、検知されたアラートはありません。」「警告」「危険」）に完全に統一しました。
- 右上のアウトラインボタンのスタイルクラスを `border-gray-300 text-gray-700 hover:bg-gray-50 transition` に統一し、不自然な枠線色の差異やフォントの太さのばらつきを解消しました（アラート履歴の戻るリンクも同様のアウトラインボタンへ統一）。
- 管理者検証パネル（Verification.jsx）で使用されていた絵文字（`⚡`, `📥`, `🌊`, `⛅`, `⚠️`, `📈`, `💡`）を、すべて洗練された Lucide-like な SVG アイコン（Zap, Inbox, Droplet, Sun, AlertTriangle, TrendingUp, Lightbulb）に置き換え、モダンでクリーンなデザインへ統一しました。

## 検証手順
1. 一般ダッシュボード（`localhost:8000`）を開き、左上が「波アイコン ＋ kawa-watch ＋ 一般ダッシュボード バッジ」になっていることを確認します。
2. 一般ダッシュボードの右上のボタンが「アラート履歴」および「検証モード」または「管理者ログイン」に日本語化されており、枠線が薄いグレーであることを確認します。
3. 「アラート履歴」ボタンをクリックして遷移し、ヘッダー左上が「波アイコン ＋ kawa-watch ＋ アラート履歴 バッジ」になっていること、右上ボタンが「ダッシュボードへ戻る」（アウトラインボタン）になっていること、およびリスト内テキスト（「検知日時」「水位」「警告/危険バッジ」）が日本語になっていることを確認します。
4. 検証モードへ遷移し、左上が「波アイコン ＋ kawa-watch ＋ 管理者検証パネル バッジ」に変更されていることを確認します（シールドアイコンが廃止されていること）。
5. 各画面のアクションボタンのホバー時の挙動（フェード効果）が、`transition` クラスの適用によりスムーズになっていることを確認します。
