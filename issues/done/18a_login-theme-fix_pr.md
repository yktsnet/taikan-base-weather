## 変更内容
- ログイン画面 `Login.jsx` の全体のテーマを、一般ダッシュボード（`bg-gray-100` や白ベースのライトテーマ）に適合したライトテーマへとデザイン修正しました。
- 背景を `bg-slate-900` から `bg-gray-100` に変更し、カードを `bg-white shadow rounded-lg border border-gray-200` に変更。
- テキストやフォームの各インプット欄、ボタンの配色も、ダッシュボードと同様のライトトーンおよびブランドカラーのインディゴ（`bg-indigo-600 hover:bg-indigo-700`）へと統一しました。
- デモアカウントの注記エリアも、ライトテーマに適した爽やかなブルー（`bg-blue-50 border-blue-100 text-blue-800`）に調和させました。

## 静的確認結果
- フロントエンドアセット（React/Inertia/Tailwind）について、`npm run build` を実行してエラーなく正常に Vite ビルド（本番ビルド）が通ることを検証しました。

## 検証手順
user様は以下の手順で動作をご確認いただけます。

1. ログイン画面（`/login`）にアクセスし、背景がダッシュボードと同じライトグレーになり、白のカードとインディゴ系のボタンで調和されたデザインに変更されていることを確認します。
2. フォームやデモアカウント注記エリアの文字が見やすく、ダッシュボードとデザイン面での連続性（同じサイト感）が保たれていることを確認します。
