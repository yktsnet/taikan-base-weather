## 変更内容
- 左上のシールドアイコンを廃止し、一般ダッシュボードおよび管理者検証パネルで共通の「波（Waves）のSVGアイコン」に変更しました。
- 一般ダッシュボードの左上にもアプリロゴ「kawa-watch」および役割バッジ「一般ダッシュボード」を追加し、管理者検証パネルとのデザイン一貫性を確保しました。
- 一般ダッシュボードに表示されるテキストやボタン（`Alert History`、`Verification Mode`、`Admin Login`）を日本語（「アラート履歴」「検証モード」「管理者ログイン」）に統一しました。
- 右上のアウトラインボタンのスタイルクラスを `border-gray-300 text-gray-700 hover:bg-gray-50 transition` に統一し、不自然な枠線色の差異やフォントの太さのばらつきを解消しました。
- 管理者検証パネル（Verification.jsx）で使用されていた絵文字（`⚡`, `📥`, `🌊`, `⛅`, `⚠️`, `📈`, `💡`）を、すべて洗練された Lucide-like な SVG アイコン（Zap, Inbox, Droplet, Sun, AlertTriangle, TrendingUp, Lightbulb）に置き換え、モダンでクリーンなデザインへ統一しました。

## 静的確認結果
- Viteによるアセットビルド（`vite build`）が正常に終了することを確認しました。
- 変更ファイル一覧:
  - `src/resources/js/Pages/Dashboard.jsx`
  - `src/resources/js/Pages/Admin/Verification.jsx`

## 検証手順
1. 一般ダッシュボード（`localhost:8000`）を開き、左上が「波アイコン ＋ kawa-watch ＋ 一般ダッシュボード バッジ」になっていることを確認します。
2. 一般ダッシュボードの右上のボタンが「アラート履歴」および「検証モード」または「管理者ログイン」に日本語化されており、枠線が薄いグレーであることを確認します。
3. 検証モードへ遷移し、以下の絵文字がSVGアイコンに変わっていることを確認します。
   - 負荷シミュレーター: ⚡（黄色のイナズマアイコン）
   - SQSキュー監視: 📥（青色のインボックスアイコン）
   - 水位データキュー: 💧（青色の水滴アイコン）
   - 気象データキュー: ☀️（黄色の太陽アイコン）
   - デッドレターキュー: ⚠️（赤色の警告三角アイコン）
   - DB書き込みパフォーマンス: 📈（緑色のトレンドグラフアイコン）
   - 豆知識欄: 💡（黄色の電球アイコン）
4. 各ボタンのホバー時の挙動（フェード効果）が、`transition` クラスの適用によりスムーズになっていることを確認します。
